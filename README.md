# StudyHub

StudyHub is an AI-Powered study tool to help students learn productively.

Uses the power of AI to help users understand subjects clearly and concisely and drive learning with hands-on practice.

- Calls the Gemini API to generate study content using user's original study notes.
- Created with JS React.
- Integrated with Firebase Firestore + Authentication.

## Key Features:
## 1. **Practice Quizzes**
Take AI generated Practice Quizzes to meet test day with confidence

## 2. **Flashcards**
Strengthen knowledge with Flashcards created with AI

## 3. **Enhanced Notes**
Study AI Enhanced Notes to efficiently understand the subject
